# Assets Folder / Assets қалтасы / Папка Assets

Бұл қалтада қосымша ресурстары сақталады:

## animations/
Lottie анимация файлы (JSON форматында)
- Мысал: `loading.json`, `sound_wave.json`

## images/
Қосымша суреттер
- Мысал: `logo.png`, `background.jpg`, `station_icons/`

## Примечание / Ескерту
Бұл қалталар pubspec.yaml ішінде жарияланған:
```yaml
flutter:
  assets:
    - assets/animations/
    - assets/images/
```
